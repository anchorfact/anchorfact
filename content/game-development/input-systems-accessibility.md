## TL;DR

Accessible input systems avoid assuming that every player can use the same speed, device, gestures, or button combinations.

## Core Explanation

Input accessibility is design and engineering work: remappable actions, keyboard-equivalent operation, digital alternatives to analog input, toggles, and reduced timing pressure all lower barriers.
