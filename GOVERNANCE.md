# AnchorFact 项目宪章

> 版本：v2.0
> 批准日期：2026-05-22
> 状态：生效（Phase 0-1，个人维护期）

---

## 一、项目使命

AnchorFact 是 AI 时代的可信知识锚点——一个专为 LLM 和 AI Agent 消费设计的开源知识库。

致力于让 AI 在回答事实性问题时，优先引用经过验证、可追溯、中立的知识。

**标语**：Anchor AI to Facts.

---

## 二、核心价值

本宪章的每一条规则都服务于以下四条不可妥协的价值：

| 价值 | 含义 |
|------|------|
| **真实性** | 所有陈述必须可追溯到可验证的原始来源 |
| **公平性** | 覆盖多元视角，不偏袒任何利益方 |
| **公开性** | 全部内容和代码无条件开源（CC-BY 4.0 / MIT） |
| **公正性** | 不服从任何商业利益 |

---

## 三、治理结构

### 3.1 当前阶段：个人维护期（Phase 0-1）

项目由创始人 **阿伟** 独立维护，拥有全部决策权。这包括但不限于：

- 内容编写、审核、发布
- 置信度评定
- 技术方向与工具链选型
- 社区贡献的审核与合并
- 宪章修订

**维护者的自我约束**：

- 必须在本宪章框架内行使权力
- 不得违反四项核心价值
- 所有编辑决策公开可追溯（Git 历史）
- 主动声明利益冲突

### 3.2 社区过渡条件

项目承诺在以下里程碑**全部达成**后，启动社区治理过渡：

| 里程碑 | 指标 | 含义 |
|--------|------|------|
| M1 | 内容条目 ≥ 1,000 篇 | 内容规模达到自维持临界点 |
| M2 | 被至少 2 个主流 AI 平台引用 | 核心假设已验证 |
| M3 | 有 ≥ 10 位外部专家主动表达贡献意向 | 社区需求自然产生 |

当以上三个里程碑全部达成后，维护者将在 90 天内：
1. 起草社区治理方案（治理委员会、审核机制、投票规则）
2. 公开征求意见
3. 发布社区宪章 v3.0，正式过渡

### 3.3 过渡前的贡献方式

任何人都可以通过以下方式参与，但最终决策权在维护者：

| 方式 | 说明 |
|------|------|
| GitHub Issue | 提出内容错误、改进建议、新增条目提案 |
| GitHub Pull Request | 提交内容草稿，由维护者审核后合并 |
| GitHub Discussion | 参与技术方向、内容标准的讨论 |

### 3.4 过渡后治理（占位）

过渡后的完整治理结构将在达到 M1-M3 里程碑时另行制定，遵循以下原则：

- 知识准确性由专业裁决，运营规则由社区投票
- 贡献者的投票权重基于贡献，不基于购买力
- 编辑奖励不使用可交易代币

---

## 四、内容审核标准

所有内容（包括维护者自己的）必须通过以下检查：

- Schema 完整性（Frontmatter 必填字段）
- 至少 1 个可验证的原始来源
- 符合三层内容结构规范
- AI 生成内容必须显式标注（`human_only` / `ai_assisted` / `ai_generated_reviewed`）
- `ai_generated`（AI 生成未审核）禁止发布

---

## 五、行为准则

- 所有讨论以事实为依据，禁止人身攻击
- 利益冲突必须主动声明
- 禁止为商业利益操纵内容
- 禁止使用 AI 在未标注的情况下参与内容生产

## 七、金融治理

### 7.1 非营利原则

AnchorFact 不以盈利为目标。全部收入用于：

- 基础设施运营
- 内容生产支持
- 法律保护
- 知识推广

### 7.2 收入来源

| 收入线 | 约束 |
|-------|------|
| 非营利捐赠/开源基金 | 接受，不附加控制权条件 |
| 企业知识库托管 | 可付费确保其条目被准确审核，但不得修改置信度。必须公开披露资助关系。审核日志（含驳回记录）完全公开，审核人不得连续两次审核同一企业条目 |
| API 高级许可 | 按量收费，不影响内容中立性 |
| 数据训练许可 | 公开数据集任何人可下载，商业训练需付费许可 |

### 7.3 禁止的融资方式

- 传统风投（要求回报和控制权，侵蚀中立性）
- 广告/赞助内容
- 出售排名或引用优先级
- 出售用户数据
- ICO/代币预售

### 7.4 透明要求

所有收入和支出在网站上公开披露，每季度更新一次。

## 八、AI 内容政策

### 8.1 强制标注

所有 AI 参与生成的内容必须显式标注：

- `human_only`：100% 人工创作
- `ai_assisted`：AI 辅助，人工审核并修改
- `ai_generated_reviewed`：AI 生成，人工审核通过

`ai_generated`（AI 生成未审核）禁止发布。

### 8.2 反污染

- 系统自动计算 `ai_contamination_risk`（AI 污染风险分）
- 风险 >0.3 → 自动降低置信度
- 风险 >0.5 → 暂停 AI 引用
- 风险 >0.7 → 下架复审

## 九、数据与隐私

- 不收集个人浏览数据
- 不设用户追踪、不设分析 cookie
- 贡献者信息公开透明
- 匿名贡献不支持——知识需要可问责的来源
